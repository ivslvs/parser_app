 This parser is written in Python using the micro framework Flask (RESTful).

- Python 3.8
- Flask
- Docker
- Celery

## Application description
User sends random url and receives all html tags.



## Quick start

```sh
docker-compose build
docker-compose up

docker-compose down
```
